# concept_library – Bibliothèque de concepts

Ce dossier regroupe des **concepts réutilisables** (workflows, méthodologies, templates) pour le travail avec les assistants IA.

## Contenu

| Concept | Dossier | Description |
|---------|---------|-------------|
| **PRP Flow** | `cc_PRP_flow/` | Product Requirement Prompts : définition du PRP, différence avec le PRD, workflow de création de PRPs, template et PRPs générés. Voir `cc_PRP_flow/README.md`. |

## Usage

- Les concepts servent de **référence** pour générer ou valider des livrables (ex. un nouveau PRP).
- Chaque concept peut contenir un README explicatif et des artefacts (templates, exemples) dans un sous-dossier dédié.
