# Contexte Gemini – Index et prompts

Ce dossier centralise ce qu’il faut fournir à l’IA (Gemini, Claude, etc.) pour travailler sur LearnSup.

## Fichiers

| Fichier | Rôle |
|---------|------|
| **[00-index-documents.md](00-index-documents.md)** | Liste de tous les documents ai_docs avec chemin et description ; ordre de lecture recommandé. |
| **[01-prompts-exemples.md](01-prompts-exemples.md)** | Exemples de prompts par cas d’usage (implémentation, E2E, nouveau PRP, conformité PRD). |

## Usage rapide

1. Pour **savoir quels docs charger** : ouvrir [00-index-documents.md](00-index-documents.md).
2. Pour **formuler un prompt** selon la tâche : ouvrir [01-prompts-exemples.md](01-prompts-exemples.md).
3. Point d’entrée général (résumé + liens) : [../gemini-context.md](../gemini-context.md) à la racine de ai_docs.
